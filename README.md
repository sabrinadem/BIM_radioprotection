Afin d'ouvrir l'application, exécuter "APP.py"
Pour ajouter des sources de radioisotopes et des matériaux, modifier "sources.py" et "composite_physics.py" respectivement. 
Pour ajouter un spectre Spekcalc, insérez un fichier .spec dans le dossier "spekcalc".
